# 📚 Book Haven - Online Book Store

Welcome to **Book Haven**, a modern and responsive online bookstore website built to provide book lovers with an amazing reading experience.

## ✨ Features

* 📖 Browse books by categories
* 😂 Funny Books Section
* 👻 Horror Books Section
* 🏰 Fictional Books Section
* 🔍 Search functionality
* 📱 Fully Responsive Design
* 🎨 Attractive and User-Friendly UI
* ⚡ Fast and Smooth Navigation

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**
* **Bootstrap** (Optional)

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/book-haven.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
